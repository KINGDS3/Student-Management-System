# Student-Management-System

A console-based Student Management System developed using Java and Object-Oriented Programming (OOP) concepts.

## Features

* Add Student Records
* Display All Students
* Search Student by ID
* Update Student Marks
* Delete Student Records
* Duplicate ID Validation
* Exception Handling for Invalid Inputs

## Technologies Used

* Java
* OOP (Object-Oriented Programming)
* ArrayList
* Exception Handling

## Sample Menu

1. Add Student
2. Display Students
3. Search Student
4. Delete Student
5. Update Student Marks
6. Exit
